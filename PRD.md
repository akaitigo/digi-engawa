# PRD: Digi-Engawa（デジタル縁側）

## 概要
公民館・地域サロンで開催する「デジタル縁側」教室の運営を支援するプラットフォーム。
大きなフォント・やさしい日本語・音声ガイド付きのステップバイステップ教材を自動生成。
受講者のつまずきポイントをサポーター側にリアルタイム可視化するヘルプボードを搭載。

## ユニークポイント
「縁側で隣の人に聞く」感覚をデジタル化し、シニアのつまずきをリアルタイムで隣のサポーターに可視化する双方向教室。

## 技術スタック
| レイヤー | 技術 |
|---------|------|
| フロントエンド | TypeScript / Next.js (PWA) |
| バックエンド | Go (API) |
| データベース | PostgreSQL |
| リアルタイム | WebSocket |
| 音声 | Web Speech API |
| インフラ | GCP Cloud Run |
| 通知 | FCM (Firebase Cloud Messaging) |

## ターゲットユーザー
- **受講者**: 60代以上のデジタル初心者。スマートフォン・タブレットの基本操作を学びたい
- **サポーター**: ボランティアスタッフ。受講者の隣で操作を支援する
- **主催者**: 公民館・地域サロンの教室運営者

## MVP機能

### F1: やさしい日本語＋音声読み上げ付きステップバイステップ教材ビューア
- [x] 教材を1ステップずつ大きなフォントで表示
- [x] 各ステップに音声読み上げボタン（Web Speech API）
- [x] やさしい日本語（漢字にふりがな、短い文）で教材テキスト表示
- [x] 前へ/次へ ナビゲーション（大きなボタン）
- [x] 進捗バー表示

### F2: 受講者の「困った」ボタンとサポーター向けヘルプボード
- [x] 受講者画面に「困った」ボタン（目立つ大きなボタン）
- [x] 押すと現在のステップ情報付きでヘルプリクエスト送信（WebSocket）
- [x] サポーター画面にリアルタイムヘルプボード表示
- [x] ヘルプリクエストの対応済み/未対応ステータス管理

### F3: 教室開催管理ダッシュボード
- [x] 教室の作成（日程・場所・定員・使用教材）
- [x] 参加者登録（受講者・サポーターの紐付け）
- [x] 教室一覧・詳細表示
- [x] 参加者への教材配布（教室コードでアクセス）

## アーキテクチャ
```
[Next.js PWA] ── WebSocket ──> [Go API] ──> [PostgreSQL]
     │                              │
     └── Web Speech API             └── FCM (通知)
```

### ディレクトリ構成
```
web/          # Next.js フロントエンド (PWA)
api/          # Go バックエンドAPI
db/           # マイグレーション・スキーマ
docs/         # ADR・設計ドキュメント
```

## 非機能要件
- アクセシビリティ: WCAG 2.1 AA準拠（フォントサイズ最小18px、コントラスト比4.5:1以上）
- レスポンシブ: タブレット・スマートフォン対応（タッチターゲット最小48px）
- PWA: オフラインで教材閲覧可能
- パフォーマンス: ページロード3秒以内（3G回線）

## スコープ外（v1.0）
- 教材の自動生成AI機能
- 多言語対応（日本語のみ）
- 管理者向け分析ダッシュボード
- 外部LMS連携
